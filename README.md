# Sistema de Gestión de Personas

Proyecto desarrollado en Java NetBeans con Oracle 10g.

## Funcionalidades
- Registrar personas
- Consultar personas
- Actualizar registros
- Eliminar registros
- Reportes
- Exportación CSV

## Tecnologías
- Java
- Swing
- Oracle 10g
- JDBC
- NetBeans
